# Security

## Reporting vulnerabilities

To report a security issue, please email **security@example.com** (replace with your contact). We will respond and work on a fix as appropriate.

## Data

- Video and analytics data are stored securely.
- We do not share your data with third parties for marketing or unrelated purposes.

## No secrets in repo

- Do not commit secrets, API keys, or credentials to the repository.
- Use environment variables and `.env` (ignored by git) for local configuration.
- In CI/CD, use your platform’s secret management for production credentials.
